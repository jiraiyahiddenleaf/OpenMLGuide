# Quantum Machine Learning Papers


- [PennyLane: Automatic differentiation of hybrid quantum-classical computations](https://arxiv.org/abs/1811.04968v4) (2018): A Python framework, facilitates quantum and hybrid quantum-classical computation optimization and ML. It offers compatibility with qubit and continuous-variable models, enabling gradient calculations for variational quantum circuits, bridging classical and quantum techniques in optimization and machine learning. [(code)](https://github.com/PennyLaneAI/pennylane)

- [TensorFlow Quantum: A Software Framework for Quantum Machine Learning](https://arxiv.org/abs/2003.02989v2) (2020): An extension of TensorFlow, empowers quantum ML. It supports model construction, training, quantum circuit simulation, and combines classical and quantum computing within a single model, offering a high-level interface for hybrid quantum-classical models. [(code)](https://github.com/tensorflow/quantum)

- [A divide-and-conquer algorithm for quantum state preparation](https://arxiv.org/abs/2008.01511v2) (2020): Suggests a quantum state preparation method employing a divide-and-conquer algorithm to minimize the number of quantum gates needed. It involves recursively breaking down the target state into smaller sub-states that require fewer gates for preparation. [(code)](https://github.com/qclib/qclib)

- [Quantum Neuron: an elementary building block for machine learning on quantum computers](https://arxiv.org/abs/1711.11240v1) (2017): Introduces a quantum neuron as a fundamental element for ML. Utilizing a quantum circuit, it conducts nonlinear input data transformations, exhibiting promise in solving classification problems while addressing scalability challenges for larger models.

- [q-means: A quantum algorithm for unsupervised machine learning](https://arxiv.org/abs/1812.03584v2) (2019): Suggests "q-means," a quantum adaptation of the widely-used k-means clustering algorithm. It employs quantum phase estimation to determine eigenvalues from a distance matrix between data points. The study showcases q-means' potential in clustering and outlines challenges in its implementation on near-term quantum hardware.
 
### Key Papers

- [Quantum Machine Learning](https://arxiv.org/abs/1611.09347) (2014) - Early paper outlining basic goals and approaches for quantum ML.
- [Quantum algorithms for supervised and unsupervised machine learning](https://arxiv.org/abs/1307.0411) (2013) - Proposes quantum algorithms for supervised classification and unsupervised clustering.
- [Quantum Neural Networks](https://axon.cs.byu.edu/papers/ezhov.fdisis00.pdf) (2018) - Describes a quantum version of neural networks with quantum circuits.
- [Quantum variational autoencoder](https://arxiv.org/abs/1802.05779#:~:text=Variational%20autoencoders%20(VAEs)%20are%20powerful,quantum%20Boltzmann%20machine%20(QBM).) (2017) - Implements a variational autoencoder using parametrized quantum circuits.
- [Quantum Boltzmann Machine](https://arxiv.org/abs/1601.02036) (2018) - Proposes a generative quantum model based on Boltzmann machines.
- [Quantum generative adversarial networks](https://arxiv.org/abs/1804.08641) (2018) - Formulates a quantum version of GANs using quantum circuits.
- [Quantum graph neural networks](https://arxiv.org/abs/1909.12264) (2019) - Applies quantum circuits for graph representation learning.
- [Quantum algorithms for topological and geometric analysis of data](https://www.nature.com/articles/ncomms10138) (2016) - Presents quantum algorithms for data topology/geometry.
- [Quantum reinforcement learning](https://arxiv.org/abs/0810.3828) (2008) - An early approach to quantum reinforcement learning using superposition of states.
- [An introduction to quantum machine learning](https://arxiv.org/abs/1409.3097) (2014)
- [Distributed secure quantum machine learning](https://www.sciencedirect.com/science/article/abs/pii/S2095927317303250) (2017)
- [Alchemical and structural distribution based representation for universal quantum machine learning](https://pubs.aip.org/aip/jcp/article/148/24/241717/961132/Alchemical-and-structural-distribution-based) (2018)
